# Getting Started
### Prerequisites:
Node.js (Node.js) v18 or higher

MongoDB (Database) Atlas account or local instance

NPM (Node Package Manager)

## Installation
1 Clone the repository (Repo)

2 Install dependencies

## setup env variable:
PORT=

MONGODB_URI=

JWT_SECRET_KEY=

for nodemailer

CLIENT_ID=

CLIENT_SECRET=

REFRESH_TOKEN=

EMAIL_USER=


## start server
### npm run dev
